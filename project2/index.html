 Submission Requirements

Please complete the following modifications:

  1. Change the color of the header from hexidecimal purple (9933cc) to another color. Use the W3School.com's HTML Color Picker as a reference if you are unfamiliar with hexadecimal colors.
  2. Add your first and last name to the header.
  3. Change the footer to "References available upon request."
  4. Change the navigation to sections you would find on a resume (6-8 sections of your choice).  Education, Skills, Awards, Publications, Organizations are some examples of resume sections.
  5. Add a short career objective.
  6. Add at least three degrees in the aside box on the right side of the page. Include high school to fill in the area if you have less than 2 degrees.
  7. Change the breakpoint for the desktop version by at least 10 pixels in either direction.
  8. Add a Media Query to set an initial background color and change it a different color when it reaches the desktop breakpoint you just changed in #7.
  9. Change the padding of the navigation menu.
  10. Change the font size of the text in the aside box.

Note: Your modifications must alter the code, but not in a way that negatively affects the responsiveness or visual aesthetics (e.g. color combinations). This will require you to experiment with different modifications until you find changes that work well.

 Grading
